## ⚙️ Installation

Clone the repository


```bash
git clone https://github.com/D-Sourab56/My_Website.git
```

Go into the project folder

```bash
cd My_Website
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
